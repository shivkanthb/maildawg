Mail dawg is a simple service for sending emails from your app built on StdLib.

Usage

To use the mail dawg service for your own app, simply create a service from the source to your stdlib account. 

const lib = require('lib')

let data = {
  to: "bob@gmail.com",
  type: "welcome",
  subject: "Welcome to Acme"
}
lib({ bg: true }).shivkanthb.maildawg.send(data, function(err, response) {});
